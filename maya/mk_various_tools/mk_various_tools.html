Various Tools¶

機能¶

UI編集用ツール、レンダリングウィンドウリセットツール

ViewPortTool¶

  1. AllCloseWindow：

    スクリプトで開いているウィンドウ、アウトライナ等フローティングしている 全ウィンドウ閉じます

  2. ViewWindow

    別ウィンドウで下画像の新規ビューを開きます。 カメラ保存、viewport設定の保存が可能です。

RenderResetTool¶

  1. Scene Reload

    Boxに何も入っていない場合、シーン情報が読み込まれていません。 ボタンを押してReloadしてください。

  2. Save as a new scene under the following name

    シーンをリセットした後のファイル名を決定します。

  3. Reset the Scene

    選択しているノードを上Boxのファイル名で新規シーンとして保存します。

    以下各種設定をリセットします。

    1. メンタルレイ、ソフトウェアレンダー等のウィンドウが壊れた際

    2. レンダーレイヤー作成ツールが途中で止まる際、修復します。

    3. レンダリングが止まる。

警告

ファイルを選択しているかどうか、上記画像の確認メッセージを表示します。 書き出したいモデルを選択していればをYESを押してください。

ModelRestTools¶

  1. normalReset：

    法線情報（ハードエッジ、スムースエッジ）を保ったまま、モデル情報をリセットします。

ReferenceTools¶

Reference Name Space Rename¶

リファレンスネームスペースを変更します。

  1. Reference Model list Reload

    Boxに何も入っていない場合、シーン情報が読み込まれていません。 ボタンを押してReloadしてください。

  2. Rename Under Box's Name

    変更したい名前を入力してください。

  3. Reference Model NameSpace Rename

    Rename Under Box's Nameの名前にリファレンスネームスペース、ノード名を変更します。
