BackBurner Set¶

機能¶

背景レンダーレイヤー作成、Rendaring設定を行うことが出来ます。 CTRL_LightCustomに設定がある場合、lightset(morning、morningライト有り、night、nightライト有り) の設定が可能です。

注釈

ガーデンマリオのレンダリングに関する詳細は、以下テキストの確認をお願い致します。 G:/01_PM/workflow/ガーデンマリオレンダリング.txt

使用方法¶

  1. Select RenderLayer：

    バックバーナージョブに送信したいレンダーレイヤーを選択してください。

  2. manager name

    マネージャーの名前を変更してください。デフォルト値=okinawa-3 名前を変更して一度送信すると設定値を保存します。

  3. Frame Start

    バックバーナージョブレンダリングを開始するフレーム

  4. Frame End

    バックバーナージョブレンダリングを終了するフレーム

  5. Timeout

    ジョブがタイムアウトする時間

  6. Priority

    ジョブの優先順位

  7. Tasksize

    ジョブを分割するサイズ

警告

Lineレンダリングに関して**tasksizeを1**以上の値を使用すると以下レンダリングエラーが 発生しやすくなります。

backburner exit code 1073741819

警告

JOB名が長すぎるとWindowsの文字制限のため、エラーが出ます。 文字制限対策のため、本ツールではジョブ名が40文字以上の物は送信できないようにしております。 ジョブ名を短くしてから、再度送信を行ってください。

  1. Send To BackBurner with upper Box's Name

    Select RenderLayer：ツール内で選択しているRenderLayerをMaya Scene save nameの名前で保存しJOBを送信します。 _jobnameの名前を任意の名前に変更しください。

  2. 2 Send To BackBurner with upper Box's Name

    上のBoxに書かれている名前をもとにジョブを送信します。 propであれば、propが含まれているrenderLayer全てを送信します。 prop,KDOと区切ればpropとKDOが含まれているrenderLayer全てを送信します。
