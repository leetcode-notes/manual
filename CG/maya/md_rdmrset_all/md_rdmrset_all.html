機能¶

Material設定、Line設定、Smooth設定、Rendaring設定を行うことが出来ます。

使用方法¶

Material¶

  1. UseBacksdwoff：

    選択しているモデルにreflectivity、reflectionLimit、shadowMaskをオフにした UseBackGroundマテリアルを適用します。

  2. UseBacksdwON：

    選択しているモデルにreflectivity、reflectionLimitをオフにした UseBackGroundマテリアルを適用します。

  3. BlackSurface

    選択しているモデルに黒色のSurfaceShaderマテリアルを適用します。

  4. GreenSurface

    選択しているモデルに緑色のSurfaceShaderマテリアルを適用します。

  5. WhiteSurface

    選択しているモデルに白色のSurfaceShaderマテリアルを適用します。

  6. BlackHole

    選択しているモデルかマテリアルに、BlackHoleを設定します。 MiaMaterial、SurfaceShaderの場合はLambert materialを適応しBlackHoleの設定を行います。 RenderLayerにOverRideの設定を追加するためDefaultRenderLayerでは使用ができません。

  7. random ramp

    選択しているモデルのRamp Shaderに影の色指定用撮影 素材RGBColorを設定します。 現在割り当てられているマテリアル名にアンダーバー2つ (__)を追加し新しいマテリアルを適応します。

    注釈

    選択した順番に応じて色が割り当てられます。 同じ選択順であれば、毎回同じ色割り当てとなります。

  8. Green Mat Alpha Tex

    選択しているモデルのマテリアルに透明度テクスチャが、 使用されている場合、透明度テクスチャのついた輝度Max のランバートマテリアルを適応します。

    注釈

    選択しているモデルから下のモデル全てに適応します。

  9. Reset Material

    マテリアルが壊れている場合再設定を行います。

Line¶

  1. Select Render Camera：

    Scene内のperspカメラをリストアップします。

  2. Toon Line To Poly

    選択しているLineをポリゴン化します。

  3. Camera Connect

    ConnectするLineとCameraを選択してCamera Connectボタンを押してください。

  4. Toon Line List Window

    Lineの一覧を表示して選択することが可能です。

  5. ラインサイズ変更

    1. All

      boxの数値に応じてシーン内全てのラインをサイズ変更する事ができます。 xボタンでbox入力の必要なしに変更が可能です。

    2. Select

      boxの数値に応じて選択しているもののみサイズ変更が可能です。 xボタンでbox入力の必要なしに変更が可能です。

  6. Camera Connect

    ConnectするLineとCameraを選択してCamera Connectボタンを押してください。

  7. Line Remove

    選択しているモデルのラインを削除することが出来ます。

  8. Remove line of hiden model below from Selected Model

    選択しているモデルから下の非表示になっているモデルを、全てラインから削除します

Line Tools¶

  1. line name

    文字検索を行う事が可能です。文字を入力することで文字が含まれているラインのみがリストアップされていきます。

    ＊大文字、小文字の区別はありません

  2. Show Line,Hide Line

    上のBoxで選択しているラインの表示、非表示を行うことが出来ます。

  3. Line Set

    上のBoxで選択しているラインの設定をコピーします。このボタンを押すとCreate Line、Selected Lines Paste Settingsに設定が反映されます。

  4. Select Model

    上のBoxで選択しているラインに含まれているメッシュを選択します。

  5. Create Line

    ラインを作成します。with Defaultの場合はデフォルト設定のラインを作成します。

  6. Selected Lines Paste Settings

    ラインを設定をコピーします。with Defaultの場合はデフォルト設定のラインをペーストします。

Set Smooth and Normal Transfer¶

  1. charactor model list Reload：

    シーン内にキャラモデルが有り、何も表示されていない場合はReloadボタンを押してください。

  2. Set Smooth and Normal Transfer：

    キャラ毎のtxtデータを参照してSmoothとNormal Transferを適応します。 KDOの場合：G:/02_assets/chara/KDO/renderSet_normal.txt KDOの場合：G:/02_assets/chara/KDO/renderSet_smooth.txt

    RenderLayer 作成時も適応が可能です。

  3. Set Smooth：

    選択しているモデルにsmoothを適用します。

  4. Toon Line To Poly

    選択しているLineをポリゴン化します。

  5. Camera Connect

    ConnectするLineとCameraを選択してCamera Connectボタンを押してください。

rendaring (prefix name,camera,frame set)¶

  1. get takeNo from scenes file name：

    シーンファイル名からtakNoを取得します。下のboxに反映します。

注釈

上記のボタンだけではファイル名は変更されません。 下のSetボタンで変更を確定します。

  1. Set Image File Name Prefix：

    上記のboxの名前をレンダリングファイル名に適応します。

  2. Frame/AnimationSet+PngSet+CurentScence Frame Start and End Set：

    シーンのタイムラインをレンダリングフレームにセットします。

  3. Set Actiview Camera (RightClick Set Select Camera)

    アクティブビューポートをレンダリングカメラにセットします。 右クリックで選択しているカメラをレンダリングカメラにセットします。

rendaring (resolution)¶

  1. Change UnderBoxs 2304,1296(Default) right click 16:9 list：

    下のBoxに基本解像度の2304*1296を反映します。 右クリックで16:9リストを確認、反映することが出来ます。

注釈

上記のボタンだけではシーン解像度は変更されません。 下のSetボタンで変更を確定します。

  1. Change UnderBoxs from scenes file resolution：

    シーン解像度を下のBoxに反映します。

  2. ×1.1：

    Boxの縦、横の解像度を1.1倍に変更します。

  3. ÷1.1

    Boxの縦、横の解像度を0.9倍に変更します。

  4. Set Render Image Resolustion

    Boxの解像度をレンダリング設定に反映します。

rendaring (MentalRay Binarization,Anti Setting)¶

  1. Binarization：

    アクティブなレンダーレイヤーにキャラクターカラー設定と同じ2値化設定を行います。

  2. Anti：

    アクティブなレンダーレイヤーにキャラクターライン設定と同じアンチ設定を行います。

rendaring (Back Ground Rendaring)¶

  1. RenderLayer list Reload：

    シーン内のレンダーレイヤーが表示されていなければ、上記のボタンを押してReloadしてください。

  2. RenderLayer Select：

    レンダリングを行いたいレンダーレイヤーを選択してください。

  3. Save Name Under Box's Name：

    レンダージョブを投げる際に下のBox名でファイルを保存することができます。

  4. FrameStart：

    レンダージョブのスタートフレームを変更します。

  5. FrameEnd：

    レンダージョブのエンドフレームを変更します。。

  6. machine priority low,high'：

    レンダージョブのCPU優先度を決めることが出来ます。

  7. Back Ground Rendaring：

    選択しているレンダーレイヤーを上のBox名でファイルを保存しバックグラウンドレンダリングを行います。

注釈

レンダーレイヤージョブからレンダーレイヤーの指定、フレーム開始位置、終了位置を決定しています。 レンダーレイヤーの変更、フレームの変更でファイルを分ける必要はありません。

Blast Rendaring¶

　選択しているレンダーレイヤーを一括で 　PlayBlastを作成出来るツールです。

  1. RenderLayer list Reload：

    シーン内のレンダーレイヤーが表示されていなければ、上記のボタンを押してReloadしてください。

  2. RenderLayer Select：

    レンダリングを行いたいレンダーレイヤーを選択してください。

  3. FrameStart：

    Blastのスタートフレームを変更します。

  4. FrameEnd：

    Blastのエンドフレームを変更します。。

  5. Save Blast Images's Name：

    Blastの保存ファイル名を指定します。。

  6. viewport2.0 texture

　チェックを入れることで、以下状態でBlastを行います。

  1. アンチ最高設定

  2. 透過最高設定

  3. テクスチャオン

  4. シェーディング状態に変更

  5. viewport2.0に変更
