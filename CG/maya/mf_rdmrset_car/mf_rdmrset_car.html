Car RendSet¶

機能¶

車のレンダーレイヤー作成、マスク作成、Smooth設定、Rendaring設定を行うことが出来ます。

使用方法¶

  1. Select Cagastel：

　車、戦車などの乗り物一覧が表示されています。 　レンダーレイヤーに追加したいものを追加してください。

  1. Select Light add to sdw render layer：

    選択しているライトをsdwレイヤーに追加します。

    複数選択が可能です。

  2. Select Cmaera Camera connect linemesh

    選択しているカメラをレンダリング可能な状態にします。 ラインをポリゴンメッシュ化した後、ラインとカメラをコネクトします。

  3. RenderLayer Name

    下のBoxの名前を使用してレンダーレイヤーを作成します。 同じ名前のレンダーレイヤーがある際はそのレイヤーに追加していきます。

  4. RenderLayer Kind

    1. ibl

      通常は使用しないためデフォルトではオフにしています。

    2. COL

      ambient Lightを使用したカラーレンダーレイヤーを作成します。 Glassモデルは除外します。 シーン内のambint Lightを全て追加します。 シーン内にambient Lightがない場合は自動で作成します。

    3. SDW

      ツール内で選択しているモデルに lambetMaterialを適用してレンダーレイヤーを作成します。ツール内で選択している乗り物と、Lightを追加します。

    4. Line

      ツール内で選択しているモデルのLineをMeshにConvertし、カメラコネクトしたToonLineを作成します。またツール内で選択しているモデルにBlackHoleを適用します。

    5. Glass

      COLレイヤーを作る際に自動作成します。車をBlackHole設定にして、glassのみのレンダリングを行います。

    6. smooth+normal

      選択しているモデルに法線情報、スムース情報を追加します。

      Rendaring Toolsの Smooth Normal と同機能です。

    7. UseBackGround

      Lineレイヤーを作成する際にキャラに割り当てるBlackholeをUseBackGroundで割り当てます。

    8. smooth(all mesh)

      選択しているモデルにスムースを追加します。

    9. hide model line remove

      非表示になっているモデルを、全てラインから削除します。

      Rndaring Toolsの hide model line remove と同機能です。

  5. Make RenderLayer

    設定した状態でレンダーレイヤーを作成します。

注釈

  • モデル、カメラは必ず選択してください。

  • ライトは、選択しなくても作成が可能です。

共通部分使用方法¶

カガステル、車、キャラクターレンダーレイヤー作成ツールの共通機能です。

  1. Render Layer All Delete：

　現在作成されているレンダーレイヤーを全て削除します。

  1. Reset Render Layer：

　レンダーレイヤーエラーをリセットします。

  1. Reset Scene：

　シーンをリセットします。レンダーレイヤーの作成、レンダリングにエラーが出る際に使用してください。

VariousToolsの Reset Scene と同機能です。

  1. File Save：

　Boxの名前でファイルを保存します。

  1. Render Window Reflesh：

　レンダーウィンドウをリセットします。

  1. Add Mask Model：

　アクティブ状態のレンダーレイヤーに選択しているモデルをマスクモデルとして追加します。 　マスクモデルにはBlackHoleマテリアルを適応します。
