Django¶

Linux 開発環境にインストール¶

sudo 管理者権限で実行

sudo apt-get install build-essential

python -V

Djangoのバージョンを確認する。
$ python -m django --version

myappプロジェクトを作成する。
$ django-admin startproject myapp

$ cd myapp
$ python manage.py runserver
