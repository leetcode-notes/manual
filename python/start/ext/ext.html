ext¶

.py¶

通常のスクリプト

.py3¶

（めったに使われない）Python3スクリプト。 Python3のスクリプトは、通常、 ".py"ではなく ".py3"で終わりますが、私は数回見ました

.pyc¶

コンパイルされたスクリプト（バイトコード）

.pyo¶

optimized pycファイル（Python3.5以降、Pythonはpyoとpycではなくpycのみを使用します）

.pyw¶

ウィンドウモードで動作するPythonスクリプト、コンソールなし。 pythonw.exeで実行

.pyx¶

C / C ++に変換されるCython src

.pyd¶

Windows DLLとして作られたPythonスクリプト

.pxd¶

C / C ++ヘッダに相当するCythonスクリプト

.pxi¶

MyPyスタブ

.pyi¶

スタブファイル（ PEP 484 ）

.pyz¶

Pythonスクリプトアーカイブ（ PEP 441 ）。 これは標準のPythonスクリプトヘッダの後に バイナリ形式で圧縮されたPythonスクリプト（ZIP）を含むスクリプトです

.pywz¶

MS-Windows用のPythonスクリプトアーカイブ（ PEP 441 ）。 これは標準のPythonスクリプトヘッダの後に バイナリ形式で圧縮されたPythonスクリプト（ZIP）を含むスクリプトです

.py [cod]¶

ファイルが ".pyc"、 ".pyo"、または ".pyd"であることを意味する ".gitignore"のワイルドカード表記。
