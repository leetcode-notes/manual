数字を逆順に並べる¶

num = 12345
# 54321を求めて下さい
result = ''

while num:
    result += str(num % 10)
    num = num //10
print(result)


#####
while num: と条件を書くと、 numが0になるまで 繰り返しが実行されます。

1周目の開始時、 num は、 12345 です。これを10で割り、余りを求める（ num % 10 ）と余りは 5 です。
result += str(num % 10) と結果の 5 を文字列に変換し、 result と結合しています。

num = num // 10 では、変数 num を１０で割った商だけを変数 num に代入しています。

1周目終了時点で、変数 result は、 '5' 、 変数 num は、1234 です。

2周目で、num % 10 すると、余りは 4 です。
2周目終了時点で、変数 result は、 '54' 、 変数 num は、123 です。

3周目で、num % 10 すると、余りは 3 です。
3周目終了時点で、変数 result は、 '543' 、 変数 num は、12 です。

4周目で、num % 10 すると、余りは 2 です。
4周目終了時点で、変数 result は、 '5432' 、 変数 num は、1 です。

5周目で、num % 10 すると、余りは 1 です。
5周目終了時点で、変数 result は、 '54321' 、 変数 num は、0 です。

次の while num: の判定で、while文が終了します。
