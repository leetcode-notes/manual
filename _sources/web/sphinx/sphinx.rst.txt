MDの表示
+++++++++++
pip install recommonmark

# ファイルの拡張子に .md を追加する
source_suffix = ['.rst', '.md']
# source_suffix = '.rst' この行はコメントアウトする

# 拡張子 .md のパーサー（構文解析器）に recommonmark を指定する

.. code-block:: python

    source_parsers = {
        '.md': 'recommonmark.parser.CommonMarkParser'
    }
