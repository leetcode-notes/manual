使用方法


* マット関係ツール

.. figure:: matte_select.png
   :width: 200px

alpha matte
---------------

* 選択しているフッテージの合成モードを
  合成チャンネルに変更します。

.. note::
    複数選択可　

alpha none
---------------

* 選択しているフッテージの合成モードを
  無視に変更します。

.. note::
    複数選択可　


alpha straigt
-----------------

* 選択しているフッテージの合成モードを
  ストレートに変更します。

.. note::
    複数選択可
