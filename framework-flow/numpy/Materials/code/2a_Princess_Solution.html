1. Getting startet...one simulation¶

100 princes (applicants) ask for the Princess´ hand in marriage. The 100 princes can be ranked by quality from 1 (lowest quality) to 100 (highest quality, the "Prince Charming"). The Princess dates one prince after another (the order is random) and at the end of each Date, she has to decide irrevocably, whether to dismiss the prince or select the prince as her husband. Once she has selected one prince, the game stops.

The Princess is very picky and wants to maximise the probability of selecting Prince Charming, the prince with quality 100. All or nothing.

As the Princess is neither familiar with Python nor with math, she chooses the following naive strategy:

  • The Princess group the first 50 princes into a testgroup and the last 50 princes into a selectgroup.

  • She dismisses all princes of the testgroup and memorizes the quality of the best prince in the testgroup.

  • In the selectgroup, she selects the first prince, that has a higher quality than the best prince in the testgroup.

Alright, let´s finetune our code and we combine for the loop with if/else statements.

  • We will store the prince, that the Princess selects, in the variable selection. And before the Princess starts to date the princes of the selectgroup, the selection is set to 0 (no prince selected).
  • Then the Princess dates one prince after another and she determines for each prince, whether he is of higher quality than testgroup_best or not.
  • If he is of higher quality, the Princess selects him and the game stops (keyword break).
  • If he is of lower quality, the Princess dismisses him (keyword pass) and dates the next prince.
  • Finally print selection to display the result of her casting.

def Princess(size_applicants = 100, size_testgroup = 50, sims = 1000, seed = 114):
    
    results =[]
    random.seed(seed)
    for _ in range(sims):

        applicants_sorted = list(range(1, size_applicants+1))
        applicants_unsorted = random.sample(applicants_sorted, size_applicants)
        testgroup = applicants_unsorted[:size_testgroup]
        testgroup_best = max(testgroup)
        selectgroup = applicants_unsorted[size_testgroup:]

        selection = 0
        for i in selectgroup:
            if i > testgroup_best:
                selection = i 
                break
            else:
                pass
            
        results.append(selection)  
    prob_best = results.count(size_applicants)/sims
  
    return prob_best

def Princess(size_applicants = 100, size_testgroup = 50, sims = 1000, seed = 114, metric = "results"):
    
    results =[]
    random.seed(seed)
    for _ in range(sims):

        applicants_sorted = list(range(1, size_applicants+1))

5.2 Changing complete strategy¶

So far, our Princesses had the strategy to split the total group of applicants into a testgroup and a selectgroup.
They dismissed all princes in the testgroup. However, they memorized the quality of the best prince in the testgroup (the "reference prince") and selected the first prince in the selectgroup with higher quality than the reference prince.

Let´s now change the strategy and take the 2nd best, 3rd best, or n best prince in the testgroup as the reference prince.

  • Introduce the new parameter n_best to our function Princess() and set the default value to 1 (the best prince in the testgroup is the reference prince).

  • Make the appropriate changes in the function body!

def Princess(size_applicants = 100, size_testgroup = 50, sims = 1000, seed = 114, metric = "results", n_best = 1):
    
    results =[]
    random.seed(seed)
    for _ in range(sims):

        applicants_sorted = list(range(1, size_applicants+1))
        applicants_unsorted = random.sample(applicants_sorted, size_applicants)
        testgroup = applicants_unsorted[:size_testgroup]
        testgroup_nbest = sorted(testgroup, reverse = True)[n_best-1]
        selectgroup = applicants_unsorted[size_testgroup:]

        selection = 0
        for i in selectgroup:
            if i > testgroup_nbest:
                selection = i 
                break
            else:
                pass
            
        results.append(selection)  
    prob_best = results.count(size_applicants)/sims
    average = sum(results)/sims
    
    if metric == "results":
        return results
    elif metric == "prob_best":
        return prob_best
    elif metric == "average":
        return average
