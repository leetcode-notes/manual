matplotlib trigonometric¶

trigonometric

三角関数¶

三角関数を用いることで、周期的かつ滑らかに変化する値を扱うことができます。

三角関数とは？¶

以下のような直角三角形を考えます。

この三角形で、角度$\theta$と各辺の関係を次のように定義します。 $$\sin\theta = \frac{b}{c}$$

$$\cos\theta = \frac{a}{c}$$

$$\tan\theta = \frac{b}{a}$$

$\sin\theta$、$\cos\theta$、$\tan\theta$を三角関数といいます。 三角関数は以下の関係を満たします。

$$(\sin\theta)^2+(\cos\theta)^2 = 1$$

$$\tan\theta = \frac{\sin\theta}{\cos\theta}$$

なお、角度$\theta$の単位はラジアンです。180°をπと表します。

三角関数を実装¶

数式$y=\sin\theta$、$y=\cos\theta$をコードで実装します。

In [22]:
import numpy as np
import matplotlib.pyplot as plt

def my_sin(x):  
    return np.sin(x)  # sin(θ)

def my_cos(x):
    return np.cos(x)  # cos(θ)

x = np.linspace(-np.pi, np.pi)  # -πからπまで
y_sin = my_sin(x)
y_cos = my_cos(x)

plt.plot(x, y_sin, label="sin")
plt.plot(x, y_cos, label="cos")
plt.legend()
plt.show()

数式$y=\tan\theta$をコードで実装します。

In [20]:
import numpy as np
import matplotlib.pyplot as plt

def my_tan(x):  
    return np.tan(x)  # tan(θ)

x = np.linspace(-1.3, 1.3)
y_tan = my_tan(x)

plt.plot(x, y_tan, label="tan")
plt.legend()
plt.show()

演習:¶

数式$y=\sin\theta$、$y=\cos\theta$のグラフを様々なxの範囲で描画しましょう。

In [23]:
import numpy as np
import matplotlib.pyplot as plt

def my_sin(x):  
    return np.sin(x)  # sin(θ)

def my_cos(x):
    return np.cos(x)  # cos(θ)

x = np.linspace(-6, 6)  # 範囲を指定
y_sin = my_sin(x)
y_cos = my_cos(x)

plt.plot(x, y_sin, label="sin")
plt.plot(x, y_cos, label="cos")
plt.legend()
plt.show()
