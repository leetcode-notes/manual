matplotlib BASICS¶

BASICS

BASICS
¶

  • Matplotlib module allows for data visualization in both 2D and 3D plots.
  • Allows for animation.
  • Interactivity.

INSTALLING PACKAGES¶

Should be installed by default with the Anaconda distribution.

  • If not, go to command terminal.
    • conda install matplotlib
    • pip install matplotlib

  • PIL (Python Image Library):
    • conda install pillow
    • pip install pillow
