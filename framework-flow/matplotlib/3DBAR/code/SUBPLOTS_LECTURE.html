matplotlib SUBPLOTS_LECTURE¶

SUBPLOTS_LECTURE

SUBPLOTS
¶

  • We'll look at two methods for generating subplots:
    • subplot2grid
    • subplot

TRADE OFF¶

subplot2grid:

  - More code required, less elegant and not commonly used.

subplot:

  - Less code, more elegant, but more prone to error.
      - Value Error
      - Run Time Error
      - tight_layout method can cause a Run Time Error.






SIMPLE EXAMPLE¶
